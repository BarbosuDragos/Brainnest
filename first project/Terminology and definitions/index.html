Terminology & Definitions

  1. What do HTML and CSS stand for?
    HTML = Hypertext Markup Language ; CSS = cascading style sheets.
  2. Would you rather use HTML or CSS, for putting paragraphs of text on a webpage?
    I would rather use HTML for putting paragrahps of text in a webpage and CSS I would use it for editing the text.
  3. Would you rather use HTML or CSS, to change the font and background color of a button?
    I would rather use CSS for changing the color for a button.
  4. What is an HTML tag?
    An HTML tag is a set of characters that is used to indicate the beginning and end of an HTML element.
  5. What are the three parts of an HTML element?
    Start tag | Element content | End tag
  6. What is the purpose of the doctype declaration?
    Doctype helps the browser to identify your document as an HTML document.
  7. What is the HTML element?
    An HTML element is component of an HTML document that helps by structuring the webpage.
  8. What is the purpose of the head element?
    The purpose of the head element is to define the document title, style, scripts and other meta information.
  9. What is the purpose of the body element?
    The purpose of the body element is to contain all the contents of an HTML document.
  10. How do you create a paragraph in HTML? You can create a paragraph in HTML by typing " <p> " to open an paragrahp and " </p> " to close an paragrahp.
  11. How do you create a heading in HTML?
    You can create a paragraph in HTML by typing " <h1> " to open an heading element and " </h2> " to close an heading element.
  12. How many different levels of headings are there and what is the difference between them?
    Heading elements start frrom h1, which is the bigest heading, and it continues until h6, which is the smallest heading.
  13. What element should you use to make text bold and important?
    The element that should be used is the strong element.
  14. What element should you use to make text italicized to add emphasis to it?
    The element that should be used is the em element.
  15. What relationship does an element have with any nested elements within it?
    The relationship between nested elements is that of parent and child.
  16. What relationship do two elements have if they are at the same level of nesting?
    In this case the relationship is that of sibling.
  17. How do you create HTML comments?
    You can start a comment by typing <!-- and to end a comment by typing -->.
  18. What HTML tag is used to create an unordered list?
    To create an unordered list you need to use <ul>.
  19. What HTML tag is used to create an ordered list?
    To create an oredered list you need to use <ol>.
  20. What HTML tag is used to create list items within both unordered and ordered lists?
    To create a list item you need to use <li>.
  21. What element is used to create a link?
    The element that is used to create a link <a>.
  22. What is an attribute?
    An attribute is a component,located inside of an HTML element, that provide additional information about HTML elements.
  23. What attribute tells links where to go?
    The attribute that tells link where to go is href.
  24. What is the difference between an absolute and relative link?
    An absolute link contains all the informations necessary to locate a resorce.
    A relative link locates a resource using an absolute link as a starting point.
  25. Which element is used to display an image?
    The element that is used for displaying images is img.
  26. What two attributes do images always need to have?
    The attributes are src and alt.
  27. How do you access a parent directory in a file path?
    You can access a parent directory by typing ../.
  28. What are the four main image formats that you can use for images on the web?
    The four main image formats are JPEG, PNG, WebP and SVG.
